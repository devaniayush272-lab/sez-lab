# SEZ Lab

A fully responsive, interactive educational platform featuring 10 immersive STEM simulations.

## Features
- **10 Interactive Simulations**: Physics, Biology, Chemistry, Math, and Economics.
- **Progress Tracking**: Daily streak, total score, and completion status.
- **Modern UI**: Glassmorphism, dark/light mode, and smooth Framer Motion animations.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.
- **Educational & Game-like**: Score system based on interaction accuracy.

## Simulations Included
1. **Projectile Motion**: Explore quadratic curves and trajectory dynamics.
2. **Cell Inspector**: Discover organelles in plant and animal cells.
3. **Equation Balancer**: Balance chemical reactions using a molecular scale.
4. **Pulley & Force**: Calculate mechanical advantage and tension.
5. **Triangle Geometry**: Classify triangles and explore angle properties.
6. **Circuit Builder**: Master Ohm's Law with interactive components.
7. **States of Matter**: Observe particle behavior across different phases.
8. **Solar Explorer**: Scale the cosmos and understand orbital speeds.
9. **Logic Sandbox**: Build binary circuits with AND, OR, and NOT gates.
10. **Market Dynamics**: Analyze supply, demand, and equilibrium points.

## Tech Stack
- **React 18**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **Recharts** (Data Visualization)
- **HTML5 Canvas** (Physics Simulations)

## Folder Structure
- `/src/components/sims/`: Individual simulation components.
- `/src/lib/`: Utility functions.
- `/src/types.ts`: TypeScript interfaces.
- `/src/constants.ts`: App configuration and simulation data.

## Deployment
This app is ready for deployment on Vercel, Netlify, or any static hosting provider.
1. Run `npm run build`.
2. Deploy the `dist` folder.
